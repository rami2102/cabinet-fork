# How to Use Cabinet

Welcome to the meta-guide — the pages that teach you how to use the very thing you're reading them in. A bit like Fred and George writing an instruction manual for their own joke shop, except this one won't explode when you open it. Probably.

This workspace (Weasleys' Wizard Wheezes HQ) doubles as a working example of everything Cabinet can do. Every feature described below is demonstrated somewhere in the shop's pages, so feel free to poke around as you learn.

## What You'll Find Here

### [[Editor Tips]]
The WYSIWYG editor is where you'll spend most of your time. Learn about rich text formatting, slash commands, markdown toggle, tables, image handling, and auto-save. Everything you need to write like a proper shopkeeper.

### [[AI Editor Guide]]
Claude lives in the right-side panel and can edit your pages directly — no copy-paste required. Learn how to summon the AI, use @mentions to give it context, and craft prompts that get results. It's like having a very literate house-elf, except it asks for instructions instead of socks.

### [[Search and Navigation]]
Cmd+K is your Marauder's Map for the knowledge base. Full-text search, wiki-links between pages, sidebar drag-and-drop, and more. Find anything in the shop faster than you can say "Accio document."

### [[Version History]]
Every edit is saved to Git automatically. Browse diffs, see exactly what changed, and restore any previous version with one click. Even if you accidentally vanish an entire page, it's never truly gone — unlike Uncle Bilius at family gatherings.

### [[Apps and Repos]]
Cabinet isn't just a wiki — it can embed full web applications, link to external Git repositories, and render interactive tools right in the sidebar. The candy counter, owl post CRM, and prank lab are all live examples.

### [[Tasks and Jobs]]
A built-in Kanban board for tracking work, plus scheduled jobs that run on cron. From morning briefings to inventory alerts, Cabinet keeps the shop running even when you're testing Skiving Snackboxes.
